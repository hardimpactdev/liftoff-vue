# Tags Input

## What is this component?

The Tags Input component allows users to enter multiple items (tags) into a single input field. It often supports adding tags via Enter key or separation by commas.

## When to use this component?

- For entering categories, keywords, or email addresses.
- When multiple values are needed for a single field.

## Implementation Details

- **TagsInput**: The root container.
- **TagsInputInput**: The text input.
- **TagsInputItem**: The rendered tag.
- **TagsInputItemText**: The tag text.
- **TagsInputItemDelete**: The delete button for a tag.
