# Number Field

## What is this component?

The Number Field component is an input field designed specifically for numeric values. It supports increment and decrement controls.

## When to use this component?

- When capturing numeric input (e.g., quantity, age).
- When you need validation for numeric ranges.

## Implementation Details

- **NumberField**: The root component.
- **NumberFieldInput**: The input field.
- **NumberFieldIncrement**, **NumberFieldDecrement**: Controls for adjusting value.
