# Sonner

## What is this component?

Sonner is an opinionated toast component for Vue. It provides a simple and customizable way to show notifications.

## When to use this component?

- To show success, error, or info messages.
- As a replacement for the default Toast component if you prefer Sonner's style and API.

## Implementation Details

- **Toaster**: The provider component that renders toasts.
- **toast**: The function to trigger toasts.
