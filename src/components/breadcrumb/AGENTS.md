# Breadcrumb

## What is this component?

The Breadcrumb component displays a trail of links that indicates the current page's location within a hierarchy. It allows users to navigate back to parent pages.

## When to use this component?

- To show the current location in a site's structure.
- To provide quick navigation to higher-level pages.

## Implementation Details

- **Breadcrumb**: The root container.
- **BreadcrumbList**: The list of breadcrumb items.
- **BreadcrumbItem**: A single item in the list.
- **BreadcrumbLink**: A link within an item.
- **BreadcrumbPage**: The current page (typically not a link).
- **BreadcrumbSeparator**: The visual separator between items.
- **BreadcrumbEllipsis**: A component to handle truncation of long breadcrumbs.
