# Command

## What is this component?

The Command component is a fast, composable, unstyled command menu for Vue. It is often used to build command palettes (like Spotlight or Raycast).

## When to use this component?

- To implement a global command menu.
- For filtering lists or navigating the app via keyboard.
- As an alternative to a traditional search bar.

## Implementation Details

- **Command**: The root container.
- **CommandInput**: The input field for filtering commands.
- **CommandList**: The list of search results.
- **CommandItem**: A single command or result.
- **CommandGroup**: A group of related commands.
- **CommandDialog**: A modal version of the command menu.
