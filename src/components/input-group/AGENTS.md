# Input Group

## What is this component?

The Input Group component allows you to extend form controls by adding text, buttons, or button groups on either side of textual inputs, custom selects, and custom file inputs.

## When to use this component?

- To group buttons or labels with an input field (e.g., search button next to search bar).
- To prepend or append text to an input (e.g., currency symbol).

## Implementation Details

- **InputGroup**: The root container.
- **InputGroupInput**: The actual input field within the group.
- **InputGroupAddon**: Container for the addon content.
- **InputGroupText**: Text addon.
- **InputGroupButton**: Button addon.
