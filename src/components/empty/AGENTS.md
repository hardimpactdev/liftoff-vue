# Empty

## What is this component?

The Empty component is used to display a placeholder when there is no data or content to show. It often includes an illustration, a description, and an optional action.

## When to use this component?

- When a list or search result is empty.
- To encourage users to create content.

## Implementation Details

- **Empty**: The root container.
- **EmptyMedia**: For illustrations or icons.
- **EmptyTitle**, **EmptyDescription**: Text content.
