# Native Select

## What is this component?

The Native Select component wraps the browser's native `<select>` element. It provides consistent styling while leveraging the native dropdown behavior, which is often better for mobile accessibility.

## When to use this component?

- When you prefer native mobile pickers on mobile devices.
- For simple selection needs where a custom dropdown (like Select) is overkill.

## Implementation Details

- **NativeSelect**: The wrapper around the `<select>` element.
- **NativeSelectOption**: Wrapper for `<option>`.
- **NativeSelectOptGroup**: Wrapper for `<optgroup>`.
