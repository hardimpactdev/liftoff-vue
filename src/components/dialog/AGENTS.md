# Dialog

## What is this component?

The Dialog component is a modal window that overlays the main content. It focuses the user's attention on a specific task or information.

## When to use this component?

- To capture user input or confirmation.
- To display information that requires focus.
- For forms, settings, or alerts (though AlertDialog is preferred for alerts).

## Implementation Details

- **Dialog**: The root container.
- **DialogTrigger**: The element that opens the dialog.
- **DialogContent**: The dialog box itself.
- **DialogHeader**, **DialogFooter**: Semantic sections.
- **DialogTitle**, **DialogDescription**: Semantic text components.
