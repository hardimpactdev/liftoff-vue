# Menubar

## What is this component?

The Menubar component provides a visually persistent menu bar, similar to the top menu in desktop applications. It supports submenus, checkboxes, and radio items.

## When to use this component?

- For complex desktop-like web applications.
- To group app-wide commands.

## Implementation Details

- **Menubar**: The root container.
- **MenubarMenu**: A top-level menu item.
- **MenubarTrigger**: The button that opens the menu.
- **MenubarContent**: The dropdown content.
- **MenubarItem**, **MenubarCheckboxItem**, **MenubarRadioItem**: Interactive items.
