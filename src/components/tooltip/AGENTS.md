# Tooltip

## What is this component?

The Tooltip component displays a popup with information when an element receives focus or hover.

## When to use this component?

- To provide extra information about a control.
- To explain icon-only buttons.

## Implementation Details

- **Tooltip**: The root container.
- **TooltipTrigger**: The element that triggers the tooltip.
- **TooltipContent**: The information popup.
- **TooltipProvider**: Wraps the application or section to manage tooltip delays.
