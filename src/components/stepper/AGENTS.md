# Stepper

## What is this component?

The Stepper component conveys progress through numbered steps. It provides a wizard-like workflow.

## When to use this component?

- For multi-step forms or processes (e.g., checkout, signup).
- To show the user where they are in a process.

## Implementation Details

- **Stepper**: The root container.
- **StepperItem**: A single step.
- **StepperTrigger**: The clickable area of a step.
- **StepperIndicator**: The number or icon.
- **StepperTitle**, **StepperDescription**: Text details.
- **StepperSeparator**: Line between steps.
