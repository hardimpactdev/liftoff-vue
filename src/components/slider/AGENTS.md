# Slider

## What is this component?

The Slider component allows users to make selections from a range of values.

## When to use this component?

- To adjust settings such as volume, brightness, or price ranges.
- When exact values are less important than relative adjustments.

## Implementation Details

- **Slider**: The root component.
- **SliderTrack**: The track.
- **SliderRange**: The filled part of the track.
- **SliderThumb**: The draggable handle.
