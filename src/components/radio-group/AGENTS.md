# Radio Group

## What is this component?

The Radio Group component is a set of checkable buttons—known as radio buttons—where no more than one of the buttons can be checked at a time.

## When to use this component?

- When users need to select exactly one option from a list of mutually exclusive options.
- When all options should be visible (unlike a dropdown/select).

## Implementation Details

- **RadioGroup**: The container for the radio buttons.
- **RadioGroupItem**: An individual radio button.
