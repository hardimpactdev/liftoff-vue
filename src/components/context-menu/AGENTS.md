# Context Menu

## What is this component?

The Context Menu component displays a menu to the user when they right-click an element. It allows providing contextual actions for specific UI elements.

## When to use this component?

- To provide secondary actions for an item (e.g., copy, delete, share).
- To mimic native desktop application behavior in a web app.

## Implementation Details

- **ContextMenu**: The root container.
- **ContextMenuTrigger**: The area that triggers the menu on right-click.
- **ContextMenuContent**: The menu content.
- **ContextMenuItem**: A single action item.
- **ContextMenuCheckboxItem**, **ContextMenuRadioItem**: Toggleable items.
- **ContextMenuSub**: Submenus.
