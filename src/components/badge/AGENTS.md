# Badge

## What is this component?

The Badge component displays a small status indicator or label. It is often used to highlight status, counts, or tags.

## When to use this component?

- To show the status of an item (e.g., active, pending).
- To display a count or notification.
- To tag items with categories.

## Implementation Details

- **Badge**: The root component, typically with variants (e.g., default, secondary, destructive, outline).
