# Input OTP

## What is this component?

The Input OTP component is an accessible, one-time password input. It is designed for entering verification codes or PINs.

## When to use this component?

- For multi-factor authentication (MFA) or other verification flows.
- When you need a segmented input field for codes.

## Implementation Details

- **InputOTP**: The root container capable of handling the input logic.
- **InputOTPGroup**: A visual grouping of slots.
- **InputOTPSlot**: An individual character slot.
- **InputOTPSeparator**: A separator between groups.
