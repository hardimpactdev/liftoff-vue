# Navigation Menu

## What is this component?

The Navigation Menu component allows navigating through a website. It is accessible and responsive, supporting submenus and different styling.

## When to use this component?

- For the main site navigation header.
- For complex hierarchical navigation structures.

## Implementation Details

- **NavigationMenu**: The root container.
- **NavigationMenuList**: The list of items.
- **NavigationMenuItem**: A single item.
- **NavigationMenuTrigger**: The item that opens a submenu.
- **NavigationMenuContent**: The submenu content.
- **NavigationMenuLink**: A link item.
