# Resizable

## What is this component?

The Resizable component provides accessible resizable panel groups and layouts. It allows users to adjust the size of panels within a group.

## When to use this component?

- For IDE-like layouts with resizeable sidebars and panels.
- For split-view interfaces.

## Implementation Details

- **ResizablePanelGroup**: The container for panels.
- **ResizablePanel**: An individual panel.
- **ResizableHandle**: The handle used to resize panels.
