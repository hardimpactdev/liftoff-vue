# Pagination

## What is this component?

The Pagination component displays a list of page numbers or navigation controls to split content into multiple pages.

## When to use this component?

- When you have large datasets that need to be split.
- For search results, tables, or product listings.

## Implementation Details

- **Pagination**: The root container.
- **PaginationContent**: The container for items.
- **PaginationItem**: Wrapper for an item.
- **PaginationLink**: A page link.
- **PaginationPrevious**, **PaginationNext**: Navigation links.
- **PaginationEllipsis**: For skipped pages.
