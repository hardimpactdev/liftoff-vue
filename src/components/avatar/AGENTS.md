# Avatar

## What is this component?

The Avatar component is an image element with a fallback for representing the user. It handles the display of a user's profile picture or initials if the image is unavailable.

## When to use this component?

- To represent a user or entity in the UI.
- In lists, navigation bars, or profile cards.

## Implementation Details

- **Avatar**: The root container.
- **AvatarImage**: The component that renders the image.
- **AvatarFallback**: The component shown when the image fails to load or is not provided (e.g., user initials).
