# Aspect Ratio

## What is this component?

The Aspect Ratio component displays content within a desired ratio. It ensures that an element maintains a specific width-to-height ratio regardless of its container's size.

## When to use this component?

- When displaying images, videos, or maps that need to effectively maintain their dimensions.
- To prevent layout shifts by reserving space for media as it loads.

## Implementation Details

- **AspectRatio**: The component that wraps the content and enforces the aspect ratio.
