# Range Calendar

## What is this component?

The Range Calendar component allows users to select a range of dates. It visualizes the start and end dates and the range in between.

## When to use this component?

- For booking ranges (e.g., hotel stays, flights).
- For filtering data by date range.

## Implementation Details

- **RangeCalendar**: The root component.
- Similar structure to Calendar but supports range selection natively.
- **RangeCalendarHeader**: Navigation.
- **RangeCalendarGrid**: Grid of days.
- **RangeCalendarCell**: Individual day cell.
