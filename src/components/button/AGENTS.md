# Button

## What is this component?

The Button component displays a clickable element that triggers an action. It supports various variants, sizes, and states.

## When to use this component?

- To trigger an action (e.g., submit a form, open a modal).
- To navigate to another page (when used as a link).

## Implementation Details

- **Button**: The root component.
- **Variants**: Supports different visual styles like default, destructive, outline, secondary, ghost, and link.
- **Sizes**: Supports different sizes like default, sm, lg, and icon.
