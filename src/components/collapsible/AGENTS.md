# Collapsible

## What is this component?

The Collapsible component is an interactive element that expands or collapses a panel. It is useful for hiding content that is not immediately relevant.

## When to use this component?

- To allow users to toggle the visibility of specific content sections.
- For "show more/show less" functionality.

## Implementation Details

- **Collapsible**: The root container.
- **CollapsibleTrigger**: The element that toggles the state.
- **CollapsibleContent**: The content shown when expanded.
