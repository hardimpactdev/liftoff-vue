# Popover

## What is this component?

The Popover component displays rich content in a portal that is triggered by a button. It is lighter than a dialog but more robust than a tooltip.

## When to use this component?

- To display small settings panels or filters.
- To show extra details when clicking an element.

## Implementation Details

- **Popover**: The root container.
- **PopoverTrigger**: The element that opens the popover.
- **PopoverContent**: The content panel.
- **PopoverAnchor**: An optional anchor element.
