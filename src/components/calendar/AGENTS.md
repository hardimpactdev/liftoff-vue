# Calendar

## What is this component?

The Calendar component allows users to select a date or a range of dates. It is a reusable date picker component that can be used in forms or as a standalone element.

## When to use this component?

- To allow users to select a date (e.g., birthday, appointment).
- To allow users to select a range of dates (e.g., booking a hotel).
- To display a calendar view.

## Implementation Details

- **Calendar**: The root component.
- **CalendarHeader**: Contains the navigation buttons and the current month label.
- **CalendarGrid**: The grid of dates.
- **CalendarCell**: A single date cell.
- **CalendarCode**: Typically composed of smaller primitives for flexibility.
