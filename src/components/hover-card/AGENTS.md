# Hover Card

## What is this component?

The Hover Card component is a card that appears when a specific element is hovered. It allows users to preview content available behind a link.

## When to use this component?

- To show user profiles when hovering over an avatar.
- To show details about a link without navigating.

## Implementation Details

- **HoverCard**: The root container.
- **HoverCardTrigger**: The element that triggers the hover card.
- **HoverCardContent**: The card content.
