# Table

## What is this component?

The Table component allows you to display data in rows and columns. It handles headers, bodies, footers, and captions with consistent styling.

## When to use this component?

- To display tabular data.
- For lists of items with multiple attributes.

## Implementation Details

- **Table**: The root table element.
- **TableHeader**, **TableBody**, **TableFooter**: Standard table sections.
- **TableRow**, **TableHead**, **TableCell**: Row and cell elements.
- **TableCaption**: Caption for the table.
