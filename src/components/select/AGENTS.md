# Select

## What is this component?

The Select component displays a list of options for the user to pick fromâ€”triggered by a button. It allows customizing the appearance of the select menu beyond browser defaults.

## When to use this component?

- When you need a styled dropdown menu for selection.
- When native select styling is insufficient.

## Implementation Details

- **Select**: The root container.
- **SelectTrigger**: The button that opens the menu.
- **SelectValue**: The currently selected value display.
- **SelectContent**: The dropdown content.
- **SelectItem**: A single option.
- **SelectGroup**: Groups options.
- **SelectLabel**: Label for a group.
- **SelectSeparator**: Divider.
