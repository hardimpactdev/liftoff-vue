# Pin Input

## What is this component?

The Pin Input component allows users to enter a sequence of characters, usually for a PIN or verification code. It is similar to Input OTP but might offer different features or api.

## When to use this component?

- For entering PIN codes.
- For entering 2FA codes.

## Implementation Details

- **PinInput**: The root container.
- **PinInputGroup**: Groups slots.
- **PinInputSlot**: Individual character input.
