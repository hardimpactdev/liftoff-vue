# Sidebar

## What is this component?

The Sidebar component is a comprehensive, composable sidebar system. It supports collapsible groups, menus, and responsiveness.

## When to use this component?

- For the main application navigation sidebar.
- When you need a complex sidebar with submenus and groups.

## Implementation Details

- **Sidebar**: The root container.
- **SidebarProvider**: Provides context.
- **SidebarHeader**, **SidebarContent**, **SidebarFooter**: Main sections.
- **SidebarMenu**: Menu container.
- **SidebarMenuItem**: Individual item.
- **SidebarGroup**: Grouping of items.
