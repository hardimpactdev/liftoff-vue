# Switch

## What is this component?

The Switch component is a control that allows the user to toggle between checked and unchecked states. It is similar to a checkbox but uses a toggle switch appearance.

## When to use this component?

- For binary settings (on/off).
- When instant activation/deactivation is implied (vs submitting a form).

## Implementation Details

- **Switch**: The root component.
- **SwitchThumb**: The moving part of the switch.
