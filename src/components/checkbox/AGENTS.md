# Checkbox

## What is this component?

The Checkbox component is a control that allows the user to toggle between checked and unchecked states. It acts as a substitute for the native HTML `<input type="checkbox">`.

## When to use this component?

- To allow users to select one or more items from a set.
- To toggle a single option on or off.

## Implementation Details

- **Checkbox**: The root control.
- **CheckboxIndicator**: The visual indicator of the checked state.
