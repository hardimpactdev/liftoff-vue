# Button Group

## What is this component?

The Button Group component is used to group related buttons together. It creates a cohesive look and feel for a set of actions.

## When to use this component?

- To group buttons that are related (e.g., view toggle: list/grid).
- To create a toolbar effect.

## Implementation Details

- **ButtonGroup**: The container.
- **ButtonGroupText**: Text label within the group.
- **ButtonGroupSeparator**: Divider between buttons.
