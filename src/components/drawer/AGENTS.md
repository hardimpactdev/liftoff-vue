# Drawer

## What is this component?

The Drawer component is an overlay panel that slides out from the edge of the screen (commonly bottom or side). It is often used on mobile devices as a replacement for dialogues or for navigation.

## When to use this component?

- For mobile-first menus or actions.
- To display supplementary content without leaving the current page.
- As a sheet or bottom sheet on mobile.

## Implementation Details

- **Drawer**: The root container.
- **DrawerTrigger**: The element that opens the drawer.
- **DrawerContent**: The sliding panel.
- **DrawerHeader**, **DrawerFooter**: Semantic sections.
