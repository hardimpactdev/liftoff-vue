# Sheet

## What is this component?

The Sheet component extends the Dialog component to display content that complements the main screen content. It typically slides in from the side (like a Drawer, but often used for desktop side panels).

## When to use this component?

- To display edit forms, filters, or details without leading the page.
- For side navigation on desktop.

## Implementation Details

- **Sheet**: The root container.
- **SheetTrigger**: The element that opens the sheet.
- **SheetContent**: The panel content.
- **SheetHeader**, **SheetFooter**: Semantic sections.
