# Dropdown Menu

## What is this component?

The Dropdown Menu component displays a list of actions or options triggered by a button. It is a fundamental UI pattern for navigation bars and action menus.

## When to use this component?

- To group related actions under a single button (e.g., "more" menu).
- For user profile menus, settings menus, etc.

## Implementation Details

- **DropdownMenu**: The root container.
- **DropdownMenuTrigger**: The button that toggles the menu.
- **DropdownMenuContent**: The menu container.
- **DropdownMenuItem**: A single action.
- **DropdownMenuLabel**, **DropdownMenuSeparator**: Structural elements.
